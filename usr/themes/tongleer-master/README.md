# 主题特点
 - 模拟微博主页，适合自媒体站长使用。

 - 可自定义昵称、头像、简介、认证信息等信息

# 使用教程
 - 将本主题里的所有文件放在您网站目录的usr/themes内，注意文件夹名字必须为tongleer。

 - 后台->外观->启用本主题->进入设置外观内填写信息

 - 本主题限个人使用，公开发布请注明原作者：二呆，及链接：http://www.tongleer.com
